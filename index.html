Sobre mí

Soy desarrollador backend de Argentina con experiencia construyendo APIs REST seguras y desplegando aplicaciones en entornos Linux utilizando Docker y Nginx. Trabajo principalmente con Java y Spring Boot, y también con PHP (Laravel), desarrollando soluciones orientadas a buenas prácticas y arquitectura limpia. He participado en el desarrollo de sistemas web en producción, implementando autenticación con JWT, control de roles y documentación de APIs con Swagger. Me interesa especialmente el backend y la arquitectura de aplicaciones escalables. Además, cuento con experiencia en frontend moderno (Angular, TypeScript, Tailwind), lo que me permite comprender la aplicación de manera integral y colaborar eficientemente en proyectos fullstack.
